Facebook Ad Size Preview (Iron Drift)

Client review page showing common Meta/Facebook video aspect ratios and how one encoded creative is fitted in each placement. Canonical deliverable in this project is output.mp4 at 1080x1350 (4:5).

Facebook Feed (Recommended)

Aspect ratio: 4:5 - Common export: 1080x1350

Built and verified
  • Best mobile feed coverage among feed-safe ratios.
  • Our actual encoded output uses this size.

Facebook Feed (Also allowed)

Aspect ratio: 1:1 - Common export: 1080x1080

Allowed placement ratio
  • Preview shows the 4:5 file fitted inside a square slot.
  • If a true 1:1 file is needed, re-encode specifically to 1080x1080.

Reels / Stories style

Aspect ratio: 9:16 - Common export: 1080x1920

Allowed in vertical surfaces
  • Preview shows how the same creative is scaled/fitted into tall placements.
  • For best full-screen fill, create a dedicated 9:16 master.
